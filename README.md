# rocket_shell

#### 介绍
一款与Rocket配套的补丁增量离线部署工具

####  安装补丁包
$ rocket-shell <补丁包> -i

#### 解析补丁包到补丁包所在目录
$ rocket-shell <补丁包> -p

#### 安装补丁包同时指定配置文件
$ rocket-shell <补丁包> -c <配置文件> -i

#### 获取帮助信息
$ rocket-shell -h
